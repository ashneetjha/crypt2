Cryptarithmetic Puzzle Rules
  1. Each letter represents a unique digit (0-9)
  2. The same letter always represents the same digit throughout the puzzle
  3. Different letters must represent different digits
  4. Leading digits (first letter of any word) cannot be zero
  5. Find the digit assignment that makes the arithmetic equation true
  6. Use logic and deduction to solve the puzzle step by step

Example: If S=9, E=5, N=6, D=7, M=1, O=0, R=8, Y=2
